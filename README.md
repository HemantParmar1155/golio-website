# Golio Website

A modern, responsive React website built as part of a front-end assessment. Designed to showcase portfolio, services, and team members.

---

## **Live Demo**
[View live site](https://hemantparmar1155.github.io/golio-website/)

---

## **Overview**
Golio is a component-based website featuring:

- Hero section with CTA buttons  
- Bootstrap-inspired UI section  
- Project & service cards with hover effects  
- Testimonials & team sections  
- Trusted company logos  
- Footer with Light/Dark mode toggle  

---

## **Tech Stack**
- **Frontend:** React, Vite  
- **Styling:** CSS (component-based + global)  
- **Deployment:** GitHub Pages  

---
